Os códigos devem ser executados em sistema operacional Linux


Procedimentos para execução do código:

Comandos para Compilar
 
javac -sourcepath src/ -d bin/ -encoding UTF-8 -g src/PriorityQueueHeap/PriorityQueueMain.java




Comandos para Executar
 
java -cp bin/ PriorityQueueHeap.PriorityQueueMain



Após é exibido na tela o seguinte o menu, para a escolha de opções:

1. Demostration 	- realiza uma pequena demonstração de como funciona o programa                                           
2. Insert   		- insere um item na fila de prioridades                                           
3. Get      		- retorna o item de maior prioridade sem removê-lo                                  
4. Remove   		- remove o item de maior prioridade                                        
5. HeapSort 		- realiza o heapsort
6. Size     		- retorna o tamanho da fila de prioridades, o número de elementos armazenados
7. Is Empty 		- verifica se a fila está vazia
8. Print Queue Heap  	- imprime a fila de prioridades 
9. Read File         	- lê o arquivo               
0. Exit              	- sair
